# 🚀 JFK Evolution - Site Web Moderne

## 📋 Description

Site web professionnel et moderne pour **JFK Evolution**, expert en télécommunications à Lyon depuis 2000. Le site présente une expertise complète en installation fibre optique, maintenance réseaux et supervision de chantiers techniques.

## ✨ Caractéristiques Principales

### 🎨 Design Moderne
- **Interface élégante** avec palette de couleurs technologiques (bleus, gris, blanc)
- **Design responsive** et mobile-first
- **Typographie moderne** avec la police Inter
- **Animations subtiles** et micro-interactions

### 🛠️ Technologies Utilisées
- **HTML5** sémantique et accessible
- **CSS3** avec variables et animations
- **JavaScript ES6+** pour les interactions
- **FontAwesome** pour les icônes
- **Google Fonts** pour la typographie

### 📱 Responsive Design
- **Mobile-first** approach
- **Breakpoints** optimisés pour tous les appareils
- **Navigation mobile** avec menu hamburger
- **Images optimisées** et lazy loading

## 🚀 Installation et Utilisation

### 1. Cloner le Projet
```bash
git clone [url-du-repo]
cd jfkevolution-main
```

### 2. Ouvrir le Site
- Ouvrir `index.html` dans un navigateur web
- Ou utiliser un serveur local (recommandé)

### 3. Serveur Local (Optionnel)
```bash
# Avec Python 3
python -m http.server 8000

# Avec Node.js
npx serve .

# Avec PHP
php -S localhost:8000
```

## 🎨 Système de Design

### Palette de Couleurs
```css
/* Couleurs principales */
--primary: #1e40af;      /* Bleu principal */
--secondary: #3b82f6;    /* Bleu secondaire */
--accent: #60a5fa;       /* Bleu accent */
--success: #10b981;       /* Vert succès */
--error: #ef4444;         /* Rouge erreur */

/* Couleurs neutres */
--text: #1f2937;         /* Texte principal */
--text-light: #6b7280;   /* Texte secondaire */
--background: #ffffff;    /* Arrière-plan */
--surface: #f8fafc;      /* Surface secondaire */
```
